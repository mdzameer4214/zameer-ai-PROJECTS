## Projects

### 1. YouTube Trend Prediction
**Description:** Analyzes YouTube video data to predict which videos are likely to trend based on historical engagement metrics and content features.

**Features:**
- Data collection and preprocessing from YouTube datasets
- Exploratory data analysis (EDA) with visualization
- Machine learning models for trend prediction
- Performance metrics and evaluation

**Technologies:**
- Python, Pandas, NumPy
- Scikit-learn for ML models
- Matplotlib & Seaborn for visualization

**Key Files:**
- `dataset.csv` - YouTube video dataset
- `model.py` - Training and prediction code
- `output.png` - Visualization results

---

### 2. Student Performance Predictor

**Features:**
- Student data preprocessing
- Feature engineering and selection
- Classification/regression models
- Performance insights and reporting

**Technologies:**
- Python, Pandas, NumPy
- Scikit-learn for predictive modeling
- Matplotlib for data visualization

**Key Files:**
- `model.py` - Model training and prediction
- `output.png` - Performance visualizations

### 3. AI Chatbot
